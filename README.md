# restaurant-list

A restaurant list website built with Node.js and Express. It allows the users to search for the restaurants they want and get the detail information for each restaurant.

## Features
1. Users can view all the restaurants in the list
2. Users can search for the restaurants with keywords
3. Users can view the detail of the restaurants by clicking on the cards

# Getting Started

## Environment
1. Node.js

## Installation
1. Open the Terminal and clone the repo
```bash
git clone https://github.com/ping8601/restaurant-list/
```
2. Open the foler of the project
```bash
cd restaurantList
```
3. Install NPM
```bash
npm install
```
4. Install nodemon
```bash
npm install -g nodemon
```
5. Start the server
```bash
npm reun dev
```
6. Open the brower and enter the link http://localhost:3000 to start exploring the website!
